In this project, I worked on predicting bike rental demand based on different factors like weather conditions, time, and seasonal patterns. I explored the dataset to understand how variables such as temperature, humidity, and whether it’s a working day or holiday affect the number of rentals. After cleaning and analyzing the data, I built and compared multiple machine learning models to predict demand more accurately. This project gave me a good understanding of how real-world factors influence user behavior and how predictive models can help businesses manage resources better and improve planning.
